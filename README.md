# Recipely

A recipe discovery mobile app built with React Native (Expo Router) and Firebase Authentication.

## Features
- Splash screen & onboarding
- Sign up / Sign in / Logout via Firebase Authentication
- Firestore-backed user profiles (editable name & country)
- Browse featured & popular recipes by category
- Recipe details with ingredients and instructions

## Tech Stack
- React Native (Expo, Expo Router)
- Firebase Authentication & Firestore
- Poppins font family
